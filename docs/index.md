## 🏗️ Estrutura do Projeto

O projeto utiliza o modelo **GitOps para Governança**, onde cada documento Markdown no repositório reflete a verdade atual da gestão.

- **Atas**: Registros de todas as reuniões deliberativas.
- **Decisões**: Documentos formais que consolidam diretrizes.
- **Atividades**: Log mensal automático de tarefas e resoluções.

---
*Este portal é atualizado automaticamente via GitHub Actions.*
